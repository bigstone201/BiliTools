# BiliTools 🚀
优化bilibili的功能，如倍数设置

![License](https://img.shields.io/github/license/bigstone/BiliTools)
![Version](https://img.shields.io/badge/version-1.0.0-blue)

一个轻量级的 Edge/Chrome 浏览器扩展，旨在优化B站使用体验，如突破 Bilibili 播放器限制，解锁任意倍速播放（如 3.0x, 4.0x）。

## ✨ 功能特性 (Features)
- ⚡ **自定义倍速**：支持输入任意数值（0.1x - 16.0x）。

- 🎨 **原生融合**：UI 风格完美融入 B 站播放器菜单。

- 💾 **自动记忆**：保存你常用的倍速配置（开发中）。

- 🔒 **安全隐私**：不收集任何用户数据，仅在 B 站页面运行。

  

  本插件已实现功能：
  -自由调整自己的倍速表
  -长按右键加速的倍速可调
  -新加鼠标左键滑动视频，可以调整视频进度，可以像移动端那样轻松调整进度

## 📦 安装指南 (Installation)

### Edge商店安装

在[BiliTools - B站增强 | 16倍速 & 沉浸体验 - Microsoft Edge Addons](https://microsoftedge.microsoft.com/addons/detail/bilitools-b站增强-16倍速-/docphhncedcnmienflkdhmmdgcmgbbap)中下载

### 开发者模式安装（适用Chrome）

1. 下载本仓库的 [最新 Release](这里填你的Release链接) 或点击 `Code` -> `Download ZIP`。
2. 解压下载的文件。
3. 在 Chrome/Edge 浏览器地址栏输入 `chrome://extensions/`。
4. 打开右上角的 **开发者模式 (Developer mode)** 开关。
5. 点击左上角的 **加载已解压的扩展程序 (Load unpacked)**。
6. 选择解压后的文件夹即可。
